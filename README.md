# Fashion Product Image Classification Using Deep Learning

This project implements a deep learning model to classify fashion product images
into different categories using Convolutional Neural Networks (CNNs).

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- OpenCV / PIL
- Google Colab

## Features
- Image preprocessing and augmentation
- CNN-based multi-class classification
- Model training and evaluation

## Usage
Open the notebook in Google Colab and run all cells.
